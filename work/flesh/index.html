Flesh Overview

Flesh is comprised of 3 levels of "anatomy". The first level would be a tendon. Sitting on top of tendons, would be the muscle. Finally, above the muscle would be skin. While an element can have one of these classes assigned to obtain a level of anatomy, many specific elements have an assumed level within Flesh by default. An example would be the code elements within this overview- they are considered to be in the skin level within Flesh.
